Projects

Gerrymandering Tool

This application is used to find gerrymandering of voting precincts in states.It displays voting patterns and demographics as well as boundaries in a map. A user has the ability to edit boundaries and fix data errors or boundary errors. It uses React for the front end, Spring Boot with Hibernate, Python to process data, and Leaflet as the mapping API.

The Shield Hero

This game is written in Typescript using the Phaser game framework. A player completes a level by memorizing a colored sequence correctly. Difficulty ramps up through new mechanics in later levels. A small system was created where a level can simply be read from a JSON file.

Tabnabbing Fighter

This Chrome extension is used to detect tabnabbing attacks and send reports to a central server. A tabnabbing attack is when a malicious page in a background tab changes the screen to phish sensitive data. After a site is reported, any other visitor of the site with the extension will be alerted. It uses Bootstrap for the UI, Firebase to store blacklisted sites, and the Chrome API for screenshotting and local storage.

Graham Scan Visualizer

This tool is used to visualize the graham scan algorithm. The algorithm is used to find a convex hull of a set of points. It can take the user's points or randomized points. It uses Java and JavaFX to input data and output a graph.

Maple NoteJotter

My very first project is this application for jotting down notes or lists. A user can add different types of data to a list and it persists in a database. There is also a notepad feature, where a user can write anything, and it saves automatically. It uses MySQL, JDBC, and java/javaFX. I like to include this project to show where I started from.
